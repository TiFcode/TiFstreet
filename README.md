# TiFstreet

A touch-first city driving game for TiFgames.

## Gameplay
- 3D-style third-person view from behind the car
- On-screen joystick on the left for steering
- GO and BRAKE pedals on the right
- Slow car movement for easy control
- Traffic lights / semaphores at intersections
- Red-light violations are counted as infractions

## Controls
- **Left joystick:** steer
- **Right GO pedal:** accelerate gently
- **Right BRAKE pedal:** slow down / stop
- **Keyboard fallback:** arrows or WASD

## Notes
Built as a single-file HTML5 canvas game to keep it easy to deploy on the existing TiFgames static hosting pattern.
